## About
A command line tool, making it easier to deal with files/directories.

## Usage

**flatten**
```shell
./fu flat target_dir [new_dir] [--deep] 
```

**rename**
```shell
./fu rn target_file new_file
```

## Features

- [x] flatten folder
  - [x] flatten folder in target dir
  - [x] rename/move after flatten
  - [x] deep mode 
- [ ] batch rename files 
  - [ ] rename a file
  - [ ] batch rename files
    - [ ] files filter
    - [ ] custom name rules
